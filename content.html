Throughout the Atomic Learning Platform's pages about R, you will encounter code cells like the one below:

print("Hello, Atomic Learning!")

These code cells are designed to act as examples that you can interact with directly within the learning platform. You can run them by clicking the "run" below the code cell. Try this with the example above now (don't worry if you don't understand what the code means for now). You should see the output of the code appear just below the cell. You can also run the code by pressing shift+Enter

You can also edit the code within a code cell to experiment with variations of the example code. Try changing "Atomic Learning" in the code cell above to your name and then re-running the cell. If you want to return the code cell to its original state, simply refresh the page.
